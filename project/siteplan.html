Smart Student Money

Website Planning Document (W05 Project)

Site Name

Smart Student Money

This name was chosen because it clearly reflects the target audience (college students) and the main focus of the website—making smart decisions with money through saving and budgeting.

Optional domain availability: smartstudentmoney.org

Site Purpose

The purpose of this website is to educate college students on basic saving and budgeting principles. The site will provide clear, beginner-friendly information on how to create a budget, track expenses, build an emergency fund, and set short- and long-term financial goals. Interactive tools such as a simple budget calculator and savings goal tracker will be included to help users apply what they learn.

Scenarios

  • How can I create a monthly budget that fits my student income and expenses?
  • What is an emergency fund and how much should I save as a college student?
  • Is there an easy way to track my expenses online?

Color Schema

  • Green (#2E7D32): Used for headings, navigation areas, and primary branding to represent growth, stability, and financial health.
  • Yellow (#F9A825): Used for accents, borders, and highlights to draw attention to important information and calls to action.
  • Light Gray (#F5F5F5): Used as the page background to improve readability and contrast.

Typography

  • Poppins: Used for headings (h1–h3) to create a modern and friendly appearance.
  • Roboto: Used for body text and paragraphs for readability and accessibility.
